# Dragon Dash 🐉

ドラゴンを操作してトンネルを駆け抜ける Babylon.js 製のブラウザゲーム。
サーバー不要の完全な静的サイト。

## 🎮 プレイ（GitHub Pages）

**▶ https://ron0401.github.io/DragonDash/game/**

スマホ・タブレット・PC のブラウザでそのまま遊べます。

### 📲 アプリとしてインストール（PWA）

ホーム画面に追加すると、アドレスバーなしの全画面アプリとして起動でき、
一度読み込めばオフラインでも遊べます。

- **Android (Chrome)** : スタート画面の「📲 インストール」ボタン、
  またはブラウザメニュー →「アプリをインストール / ホーム画面に追加」
- **iPhone / iPad (Safari)** : 共有ボタン → 「ホーム画面に追加」
- **PC (Chrome / Edge)** : アドレスバー右の ⊕ インストールアイコン

### 📤 共有 & 🎴 グリーティングカード

スタート画面・ゲームオーバー画面のボタンから:

- **📤 アプリを共有** : 端末の共有シート経由で LINE・X・メッセージアプリ等へアプリ URL を共有
  （Web Share API 非対応のブラウザではリンクをクリップボードにコピー）。
- **🎴 カードを保存** : アプリ URL の QR コードとキャラクター（自機ドラゴン＋仲間たち）、
  自己ベスト／到達距離を載せたグリーティングカードを画像生成。スマホでは共有シートで
  写真に保存・LINE 送信、PC では PNG ダウンロード。QR 生成は外部依存なしの自前ライブラリ
  （`lib/qrcode.js`）なのでオフラインでも動作。

### 📸 フォトモード（カメラ AR）

スタート画面の **「📸 フォトモード」** ボタンから起動。端末のカメラ映像を背景に、
選択中のキャラクター（ドラゴンや仲間たち）を 3D で重ねて、好きな場所に“AR配置”できます。

- **🖐️ ドラッグで回転 ・ ピンチで大きさ ・ 2本指で移動** して構図を決める
- **🔄 カメラ切替** で前面／背面カメラを切り替え（自撮りも OK）
- **📸 シャッター** でカメラ映像＋3Dモデルを合成した写真を生成。スマホでは共有シートで
  写真に保存・LINE 送信、PC では PNG ダウンロード
- カメラ映像は `getUserMedia` で取得（HTTPS 必須）。WebXR ではなくカメラパススルー方式
  なので、iPhone / iPad の Safari でも動作。

## 構成

```
game/
├── index.html             ゲーム本体 (Babylon.js)
├── manifest.webmanifest   PWA マニフェスト（インストール情報）
├── sw.js                  Service Worker（オフライン対応）
├── icons/                 アプリアイコン (192/512, maskable, apple-touch)
├── lib/                   Babylon.js ランタイム + QRコード生成ライブラリ
├── bird2.glb              ドラゴンモデル（デフォルト自機）
├── models/                キャラクターモデル (6体・自機にも敵にもなる)
└── assets/                ステージ用フリー素材 (PBRテクスチャ / スカイボックス)
```

> 3Dモデルの生成パイプライン（TRELLIS や元写真、.ply / .mp4 などの中間生成物）は
> このリポジトリには含めていません（`.gitignore` で除外）。
> 本リポジトリはゲーム部分のみを管理します。

## 遊び方

`game/index.html` をブラウザで開くだけ。ローカル確認なら任意の静的サーバーでも可:

```bash
cd game
python3 -m http.server 8000   # http://localhost:8000
```

- 移動 : 矢印キー / WASD / マウスドラッグ
- 敵に3回ぶつかるとゲームオーバー。緑の💚で体力回復。
- **プレイキャラクター選択** : 自機はドラゴン＋仲間たち（全7体）から選べる。最初は
  **ドラゴンのみ** 解放されており、ほかのキャラクターはスタート画面では「🔒 ？？？」と
  表示される。最後に選んだキャラクターはブラウザ（localStorage）に保存され、次回起動時の
  オープニング演出もそのキャラクターで再生される。
- **✨ キャラクター解放** : プレイ中にときどき **かがやくアイテム** が出現。取るとランダムで
  ロック中のキャラクターが1体解放され、**ゲームオーバー画面で演出（紙吹雪＋登場アニメ）**
  とともにお知らせ。解放状況はブラウザ（localStorage）に保存され、ずっと記憶される。
- **🔥 コンボボーナス** : 敵を連続でかわすとコンボが伸び、5回ごとに距離ボーナスがスコアに
  加算される。
- **🔊 サウンド** : 効果音とBGMつき（外部音源ファイルなしの合成音）。スタート画面・
  一時停止画面の「サウンド」ボタンで ON/OFF でき、設定は保存される。
- **📖 あそびかた** : スタート画面の「あそびかた」ボタンから操作方法・ルール・隠しキャラの
  解放方法を確認できる。
- 最高スコア（自己ベスト）はブラウザ（localStorage）に保存される。
- **💀 Very Hard Mode** : 自己ベストが **10000m 以上** になるとスタート画面の
  モード選択に出現。**3倍速からスタートして 6倍速まで** 加速する高難度モードで、
  ステージは常に **亜空間のようなマップ（宇宙空間）** 固定。
- 距離に応じてステージが変化（実写ベースのフリー素材で表現）:
  - 0–1000m : 🌋 1st マグマだまり（溶岩PBRテクスチャ＋発光する割れ目、燃える空）
  - 1000–4000m : 🌿 2nd 岩の草原（草＋岩のPBR地面、青空HDRI）
  - 4000m– : 🌌 3rd 宇宙空間（天の川全天パノラマ、地面なしで星空を飛ぶ）
- 各ステージは PBR 地面マテリアル＋HDRI/パノラマのスカイボックス（IBL）で
  ライティングし、フォグ・ライト色をなめらかに切り替える。
